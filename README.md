# ExpiryWise

ExpiryWise is a full-stack Flutter application that helps users reduce food waste by tracking groceries, expiry dates, reminders, and pantry usage.

## Project Purpose

This project is built as a portfolio project to practise full-stack Flutter development while solving a real-world problem: household food waste.

## Tech Stack

### Frontend
- Flutter
- Dart

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL

### Tools
- Git
- GitHub
- VS Code
- Postman or Thunder Client

## Planned Features

- User registration and login
- Add and manage grocery items
- Track expiry dates
- View items expiring soon
- Mark items as used, wasted, or donated
- Dashboard summary
- Meal suggestions based on expiring ingredients
- Local notifications
- Backend API
- PostgreSQL database
